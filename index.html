Halfway There Rescue Redesign

UX Case Study

Halfway there rescue is a Charlotte area pet rescue organization run by a group of dedicated volunteers. they save dogs and cats from overcrowded shelters and help find them loving, happy homes through temporary foster placements and transportation to rescues in others states.

Problem Statement

The Halfway There Rescue was designed to attract potential donors and connect local residents looking for new pets. We have observed that our webpage isn't attracting many users, which is contributing to a lack of donations to our business that is inhibiting our ability to rescue local animals. How might we improve the site features to create a more inviting space to increase overall traffic and donations?

The Solution

Creating a more personalized and visually pleasing webpage for the Halfway There Rescue will improve the overall user experience. Higher site traffic will lead to a significant increase in donations which will allow Halfway There Rescue to help save more local pets from being in overcrowded shelters and find forever homes.

Project Duration
Week One

Surveys and interviews were conducted by multiple participants to provide insight on those who are seeking pets through adoption agencies. A User persona was developed based on the analytics. Multiple indirect and direct competitors were studied to gain insight on how to improve this original website as well as finding unique ways to improve user engagement.

Week Two

Based on the User Research, a Storyboard was created to build empathy within the User Journey. Website Annotations and Heuristics were studied in order to conduct a Card Sort with participants to provide insight on ways to improve website navigation. The SWOT Analysis helps map out key features to implement in the web redesign which led up to developing the overall Sitemap for the Information Architecture.

Week Three

A Low Fidelity wireframe was developed through finalizing the Site mapping. Usability Tests were conducted to help gain insight on improvements for navigation and verbiage to help make the redesign more engaging. The UI Style Guide provides the elements needed to create a more user friendly and uniquer experiences. Webpage Hierarchy and Color Accessibility were the key UI elements that were the main focus withing the redesign.
